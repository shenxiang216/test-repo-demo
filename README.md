# Test Repo Demo

这是一个测试仓库，用于演示 GitHub MCP 的功能。

## 功能特点

- 支持 Markdown 格式
- 自动化创建和更新
- 支持版本控制

## 如何使用

1. Clone 仓库
```bash
git clone https://github.com/shenxiang216/test-repo-demo.git
```

2. 安装依赖
```bash
npm install
```

3. 开始使用
```bash
npm start
```
